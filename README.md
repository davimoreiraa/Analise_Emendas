
# ETL de Dados sobre Emendas Parlamentares

Este projeto é resultado de um trabalho de coleta, tratamento, análise e visualização gráfica de dados sobre emendas parlamentares disponibilizadas publicamente. 

## Tecnologias utilizada

**Carregamento, Análise e limpeza:** Pandas e NumPy

**Visualização gráfica:** Matplotlib e Seaborn


### Fonte

Todos os dados usados são públicos, dispóniveis no link:
 - https://portaldatransparencia.gov.br/download-de-dados/emendas-parlamentares
